## 🚀 如何使用

### 最简单的方式：

```
1. 进入 new_app 目录
2. 双击 run.bat
3. 浏览器自动打开
4. 开始测试！
```

### 详细步骤：

1. **启动系统**
   ```bash
   cd new_app
   双击 run.bat
   ```

2. **浏览器打开**
   - 自动打开 https://localhost:5000
   - 如有证书警告，点击"高级" → "继续访问"

3. **开始测试**
   - 输入年龄和性别
   - 点击"开始评估"
   - 允许摄像头权限
   - 完成4个任务

4. **查看报告**
   - 自动生成完整报告
   - 显示统计数据和图表
   - 可以打印保存
